# Nirvik Acharekar — Portfolio

A custom portfolio for Nirvik Acharekar, a product-minded full-stack developer
based in Mumbai. The site pairs an editorial visual system with real project
work, responsive layouts, accessible navigation, and deliberately restrained
motion.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Quality checks

```bash
npm run lint
npm run typecheck
npm run build
```

## Stack

- Next.js App Router
- React and TypeScript
- Framer Motion
- Modern CSS with responsive and reduced-motion states
